Color Palette code snippet

All palette colors are available with "Backgournd" and "Text" color. Background classes end with -bg and text color classes end with -text

We automated contrast ratio(4.5:1) for all colors by default, so dark background elements gets light text color and light background elements gets darker text color, however this behaviour can be overridden as needed.
